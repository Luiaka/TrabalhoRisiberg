Desenvolvimento Web

Contexto Geral

Desenvolvimento Web é a área responsável por criar e manter sites
e aplicações acessadas por navegadores. Envolve tanto a parte visual quanto a lógica por trás
do funcionamento das páginas. É um dos pilares da internet moderna.

HTML

É a linguagem que forma a estrutura básica de
um site. Define o que será exibido, como títulos, parágrafos, imagens, links e listas.

CSS

Usado para estilizar o conteúdo definido pelo HTML. Com o CSS, é possível alterar cores,
fontes, tamanhos, posições e tornar o site mais bonito e organizado.

JavaScript

Linguagem de programação usada para dar vida ao site. Permite criar menus interativos, validar formulários,
carregar conteúdo sem recarregar a página, entre outras funcionalidades.
