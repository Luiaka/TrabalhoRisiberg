Redes

Contexto Geral

Redes de computadores permitem a comunicação entre dispositivos
para troca de dados, acesso à internet e compartilhamento de recursos. Sem redes, a internet como conhecemos não existiria.

Protocolos

São regras de comunicação. O TCP/IP garante que os dados
cheguem corretamente, enquanto o HTTP permite navegar por páginas web. Cada protocolo tem uma função específica.

Endereçamento IP

Cada dispositivo em uma rede recebe um número único chamado IP.
Ele funciona como um "endereço digital", identificando o equipamento na rede.

Roteadores e Switches

O roteador conecta uma rede local à internet. Já o switch organiza a
comunicação entre os dispositivos de uma mesma rede, como os computadores de uma escola.
